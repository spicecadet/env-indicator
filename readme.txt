Change the WordPress admin toolbar to indicate the current environment on WP Engine.

== Description ==

This plugin is designed to change the color of the admin toolbar and add the environment name as a visual indicator of which site you are using. There are 3 sites defined: Production, Staging and Development. 

== Installation ==

1. Upload the plugin to your plugins directory
2. Activate it on the wp-admin plugins page

== Frequently Asked Questions ==

= WP Engine has Production and Staging sites. Why is there a third environment for Development? = 

The development environment will be default for any local or development sites that you create outside of the WP Engine platform.
